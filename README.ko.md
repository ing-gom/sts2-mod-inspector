# 모드 인스펙터 (Sts2ModInspector)

**슬레이 더 스파이어 2**용으로, 한 가지를 가볍게 해주는 **읽기 전용** 모드입니다 — 두 모드가 **같은 게임 함수를 패치**할 때(Harmony 충돌, "혼자선 되는데 같이 켜면 깨지는" 흔한 원인) 알려줍니다.

부팅 후 약 2초 뒤 검사하고, 충돌이 있으면 메인 메뉴 **우측 하단**에 작은 배지를 띄웁니다(모드 로딩 안내 텍스트는 가리지 않음). 클릭하면 탭 목록이 열립니다.

- **심각** — 다중 transpiler(IL 재작성은 겹치면 대개 깨짐) 또는 다중 prefix(원본/다른 모드를 차단할 수 있음).
- **주의** — 서로 다른 모드의 prefix + postfix. 보통은 함께 잘 동작.
- 프레임워크(BaseLib / RitsuLib / KitLib)가 의도적으로 같은 init을 공동 패치하는 경우는 *주의*로 강등.

"모드를 하나씩 꺼보는" 이분탐색 대신, 충돌한 함수와 관련 모드를 한눈에 보여줍니다. 충돌은 *가능성*일 뿐 확정된 고장이 아니며, 심각도는 분류 힌트입니다.

가벼움 · 클라이언트-사이드 · **읽기 전용** — 게임플레이 변경 없음, 로드 순서/설정 파일을 쓰지 않음, 멀티플레이 동기화 영향 없음(`affects_gameplay: false`). 14개 언어 지원.

## 설치

릴리즈 zip을 받아 `Sts2ModInspector` 폴더를 게임의 `mods/` 디렉터리에 넣고 **Load with Mods**로 실행하세요. (또는 Steam 창작마당 구독)
