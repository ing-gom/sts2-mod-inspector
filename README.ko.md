# 모드 인스펙터 (Sts2ModInspector)

**슬레이 더 스파이어 2**용 **읽기 전용** 진단 모드. 부팅 후 약 2초 뒤 한 번 검사하고, 알릴 게 있으면 메인 메뉴 **우측 하단**에 작은 배지를 띄웁니다(모드 로딩 안내 텍스트는 가리지 않음). 클릭하면 탭으로 나뉜 리포트가 열립니다.

설치한 모드에 대해, 인게임 도구가 없던 세 가지를 보여줍니다:

1. **패치 충돌** — *두 개 이상의 모드*가 같은 게임 함수를 Harmony 패치하는 경우. 지금까지는 "모드를 하나씩 꺼보기"가 유일한 진단법이었습니다. 심각도로 분류:
   - **심각** — 다중 transpiler(IL 재작성은 겹치면 대개 깨짐) 또는 다중 prefix(원본/다른 모드를 차단할 수 있음).
   - **주의** — 서로 다른 모드의 prefix + postfix. 보통은 함께 잘 동작.
   - 프레임워크(BaseLib / RitsuLib / KitLib)가 의도적으로 같은 init을 공동 패치하는 경우는 *주의*로 강등.

2. **부팅이 무거운 모드** — **정적 무게**로 순위(로드 순서와 무관하게 *모든* 모드 커버): `.pck` 총 바이트 + 항목 수(게임이 부팅 시 pck 인덱스를 바이트로 읽음), DLL 크기, 적용된 패치 수. Mod Inspector보다 늦게 로드된 모드에는 실측 init 시간(ms)이 보너스로 붙고, 먼저 로드된 모드는 "미측정"으로 표기(숨김 없음).

3. **환경 상태** — 개발자 콘솔(디버그 모드)이 켜졌는지(`SettingsSave.full_console`; 켜져 있으면 보통 도전과제 비활성화), 그리고 dev/cheat/debug 성격으로 보이는 모드.

리포트 팝업은 **심각 / 주의 / 무거운 모드** 탭으로 나뉘며 **14개 언어**를 지원합니다.

모든 동작은 클라이언트-사이드 · 읽기 전용입니다 — 게임플레이 변경 없음, 로드 순서/설정 파일을 쓰지 않음, 멀티플레이 동기화 영향 없음(`affects_gameplay: false`).

## 설치

릴리즈 zip을 받아 `Sts2ModInspector` 폴더를 게임의 `mods/` 디렉터리에 넣고 **Load with Mods**로 실행하세요. (또는 Steam 창작마당 구독)
